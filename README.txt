This directory contains 650K Corpus, divided into 10K parts CSV: total-*.csv.
To train these, use MALLET and use open the BAT file and copy paste directly 
into the command prompt. The bat file is generated by train_run.py.

Training is done using Labeled LDA implementation from MALLET.

total-*.xml is the Output Diagnostics.
total-*.mallet is the input file after running the import program.
total-*.keys is the output from the Gibbs LDA Training.

stopwords.txt can be adjusted as needed.
